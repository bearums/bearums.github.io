Step 1: Install PyJs

First you need to install git. This is easy to do in the terminal by typing:

>>>sudo apt-get install git-core

Next you download the Pyjs module with

git clone https://github.com/pyjs/pyjs.git

change directories to where the module is downloaded and install:

cd pyjs
python setup.py install

That's it! Pyjs is installed. To check it's all worked OK, start up Python in the terminal and import pyjs

python
import pyjs

If nothing happens, then Pyjs is correctly installed

Let's try some examples!

Move into the examples directory within pyjs (you should already be there).
My favourite example is asteroids, so we'll play asteroids

cd examples/asteroids

Next we need to build the javascript:

pyjsbuild space.py

This will make a folder called output with a couple of different .html files.
Open up the space.html file and see how far you can get!

If something went wrong somewhere along the line, take a look at the pyjs wiki

Step 2: Install PyJsdl

This bit is much simpler. Basically all that needs to be done is download the pyjsdl module into where your game is.
It is crucial that the pyjsdl module is in the same directory as your python game that you want to put on the web.
For example I will have my game in /home/usr/games/pyjsdl_examples

This can be done by heading to the pyjsdl website, downloading the .zip file and unpacking it into the pyjsdl_examples directory.
Or you can do it all in the terminal:

First, change into this directory

cd /home/usr/games/pyjsdl_examples
wget "s3.gatc.ca/files/Pyjsdl_0.21.zip"
unzip Pyjsdl_0.21.zip

Now let's run a little test. There is a demo on the pyjsdl site called Serpent Duel.
All that needs to be done is to unzip this python file into /home/usr/games/pyjsdl_examples.
The file can be found here.
Again the .html page is compiled with the command "pyjsbuild serpentduel.py"
