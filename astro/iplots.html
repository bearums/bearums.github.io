Interactive Plots

Interactive plots are a nice way to display, explore and analyse data. Here is a selection of plots made using python and the Bokeh module

Semi-empirical mass formula :- source code
The semi-empirical mass formula attempts to explain nuclear structure using 5 energies: volume, surface, Coulomb, Asymmetry and pairing. Each of these energies is represented by a term in the formula, and each term contains an experimentally determined constant. With this plot you can vary the values of these constants to explore how the stability of nuclei are related to the different energies.
