Interactive Plots for MESA

Interactive plots are a nice way to display, explore and analyse data.

Here is a selection of plots made using my Interactive MESA Plotter (IMP for short) which can be downloaded from https://zenodo.org/record/7351351

IMP is a software that produces interactive plots from MESA history files using python and Bokeh. Interactive plots eliminate the need to endlessly reproduce plots because you want to zoom into a certain corner or have different quantities on the axes. They could also find a use in education.
